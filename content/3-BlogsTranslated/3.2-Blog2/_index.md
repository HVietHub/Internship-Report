# Secure Cloud Innovation Starts at re:Inforce 2025

*Every day, I work with security leaders who are balancing a critical challenge. On one hand, their organizations are transforming faster than ever—adopting disruptive technologies like generative AI and expanding cloud scale. On the other hand, they must maintain strong security controls and visibility in an increasingly complex environment. We all know that simply adding more tools and controls is not always sustainable. We need a different approach for **security at scale***.

---

## re:Inforce 2025 Roadmap: Security Driving Innovation

This has shaped the vision for **AWS re:Inforce 2025**. When done right, security at scale becomes a business accelerator, enabling organizations to move faster and with greater confidence in the cloud. This isn’t just philosophy; I’ve seen it proven repeatedly by our customers, and it’s what we aim to help every organization achieve.

At re:Inforce, we’ll share a vision for simplifying security at scale, drawing from our experience supporting millions of customers worldwide. We’ll explore how organizations are building **inherently resilient applications**—able to withstand modern threats while still driving innovation. I am especially excited to highlight real-world customer examples and architectural patterns that demonstrate how security can better support business goals.

## A Learning Environment Designed for Cloud Security

- We designed re:Inforce as a dedicated, in-person security event for a reason. While AWS general events are valuable, security practitioners need space to  
  - dive into implementation details,  
  - ask hard questions,  
  - and work through complex scenarios.

- At re:Inforce, you can:  
  - collaborate directly with engineers who built AWS security services,  
  - work with security partners,  
  - schedule one-on-one time with AWS leaders to address your unique security needs.

- We’ve built multiple **learning paths** tailored to each stage of your security journey. With more than **250 technical sessions**, you’ll find the right content—whether you want to automate security controls, align development and security teams, or transform security operations.

- Expect interactive workshops where you build solutions in real time, small-group deep dives, hands-on labs to test new methods, and builder sessions with AWS experts. Best of all: **70% of the content is advanced or expert level**, ensuring you get detailed, actionable guidance.
