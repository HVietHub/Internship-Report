**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

**Week 3:** [Getting started with Amazon S3, EC2 Advanced Features, and AWS Backup](1.3-week3/)

**Week 4:** [Working with Storage Gateway, Amazon FSx, and AWS Security Services](1.4-week4/)

**Week 5:** [Optimizing Costs with Lambda, Resource Tagging, and KMS Encryption](1.5-week5/)

**Week 6:** [Advanced IAM, RDS Deployment, and Database Migration](1.6-week6/)

**Week 7:** [Reviewing Security and High Availability for Midterm Exam](1.7-week7/)

**Week 8:** [Reviewing Performance Optimization and Cost Management - Midterm Exam](1.8-week8/)

**Week 9:** [Building Data Lakes and Advanced DynamoDB Design Patterns](1.9-week9/)

**Week 10:** [Global Serverless Applications and Event-Driven Architecture with DynamoDB](1.10-week10/)

**Week 11:** [Analytics on AWS - Glue, Athena, Kinesis, Redshift, and QuickSight](1.11-week11/)

**Week 12:** [Finding and Fixing Bugs in the Project](1.12-week12/)
