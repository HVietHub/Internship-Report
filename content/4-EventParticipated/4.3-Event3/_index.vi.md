# Chu Trình Phát Triển Hỗ Trợ AI

### Thông Tin Sự Kiện

**Thời gian:** 2:00 PM – 4:30 PM

## Tổng Quan Sự Kiện

Workshop khám phá cách AI generative biến đổi chu trình phát triển phần mềm thông qua Amazon Q Developer và trợ lý AI Kiro.

### Chủ Đề Chính

**Amazon Q Developer (2:15 – 3:30 PM)**
*Diễn giả: Toàn Huỳnh*

- **Chu Trình Phát Triển Hỗ Trợ AI:**
  - Lập kế hoạch & Kiến trúc: Thiết kế hỗ trợ AI
  - Phát triển: Tạo code từ ngôn ngữ tự nhiên
  - Testing: Tạo test tự động
  - Triển khai: Tạo IaC
  - Bảo trì: Tài liệu và hiện đại hóa

- **Khả Năng Chính:**
  - Tạo code với nhận thức ngữ cảnh
  - Giải thích và tài liệu hóa code
  - Hỗ trợ debugging
  - Tích hợp AWS (Lambda, CloudFormation, CDK)

- **Demo Trực Tiếp:**
  - Tạo REST API endpoint
  - Giải thích thuật toán
  - Tạo unit test
  - Tái cấu trúc code
  - Code hạ tầng AWS

**Trợ Lý AI Kiro (3:45 – 4:30 PM)**
*Diễn giả: My Nguyễn*

- **Tính Năng Cốt Lõi:**
  - Hoàn thiện code thông minh
  - Hiểu biết đa file
  - Trí thông minh cấp dự án
  - Tái cấu trúc tự động
  - Tạo tài liệu

- **Tăng Năng Suất:**
  - Viết code nhanh hơn 40%
  - Giảm 60% boilerplate
  - Ít thời gian hơn 50% cho tài liệu
  - Ít lỗi hơn 30% ban đầu

- **Demo Trực Tiếp:**
  - Hoàn thiện code thông minh
  - Tái cấu trúc đa file
  - Phát hiện và sửa lỗi
  - Tạo test
  - Tự động hóa tài liệu

## Bài Học Rút Ra

**Phát Triển Hỗ Trợ AI:**
- AI biến đổi mọi giai đoạn của chu trình phát triển
- Tự động hóa giải phóng developer cho công việc sáng tạo
- Tăng năng suất đáng kể và có thể đo lường
- Chất lượng cải thiện qua mẫu nhất quán

**Amazon Q Developer:**
- Mạnh mẽ cho phát triển tập trung AWS
- Ngôn ngữ tự nhiên sang code rất hiệu quả
- Bảo mật và phương pháp hay nhất tích hợp
- Hỗ trợ đa ngôn ngữ

**Kiro:**
- Xuất sắc trong hiểu biết cấp dự án
- Nhận thức ngữ cảnh nâng cao gợi ý
- Thao tác đa file tiết kiệm thời gian đáng kể
- Tùy chỉnh theo tiêu chuẩn nhóm

**Phương Pháp Hay Nhất:**
- Cung cấp ngữ cảnh rõ ràng trong prompt
- Review và hiểu code được tạo
- Lặp lại và tinh chỉnh gợi ý
- Duy trì nhận thức bảo mật
- Sử dụng để học và phát triển kỹ năng

**Các Bước Tiếp Theo:**
- Cài đặt Amazon Q Developer và Kiro
- Tích hợp vào quy trình hàng ngày
- Đo lường cải thiện năng suất
- Chia sẻ kiến thức với nhóm
